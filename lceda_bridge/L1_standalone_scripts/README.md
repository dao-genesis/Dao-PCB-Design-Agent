# L1 — 独立脚本 (即贴即跑, 0安装)

> 嘉立创EDA专业版 → 顶部菜单 → 高级 → 运行脚本 → 粘贴 → 运行

## 脚本目录

| 文件 | 功能 | 输出 |
|------|------|------|
| `01_环境侦察.js` | 编辑器版本/工作区/团队/工程/分屏/eda对象顶层键 | JSON 复制剪贴板 + 弹窗 |
| `02_列出全部工程.js` | 当前工作区所有工程 UUID + 元信息 | CSV 复制剪贴板 |
| `03_导出当前BOM.js` | 当前 PCB/原理图 → BOM CSV (含 LCSC 字段) | CSV 复制 + saveFile 下载 |
| `04_注入PFC原理图.js` | 1500W 图腾柱PFC 23种元件占位标记批量注入 | 5列网格布局 + 颜色分组 |
| `05_DRC全检.js` | 当前 PCB → DRC 错误清单 | CSV 复制剪贴板 |
| `06_批量改属性.js` | 按 designator 模式批量改属性 (DRY_RUN 默认开) | 改动列表 + 弹窗 |
| `99_启动桥接.js` | 连接到本机 Python 桥服务器 (`http://127.0.0.1:9907`) | 长轮询持续运行 |

## 调试模式 (推荐先开)

打开 F12 控制台 (按三次):
- 先在嘉立创EDA客户端按 F12 三次 → 控制台
- 输入: `window.location.href = 'https://client/editor?cll=debug';`
- 再 F12 三次 → 现在可以看到所有 console.log

## 常见问题

**Q: 弹出 "找不到方法"?**
A: API 版本可能差异. 在控制台输入 `console.log(Object.keys(eda).sort())` 查看可用对象.

**Q: 剪贴板复制失败?**
A: 浏览器权限问题. 直接在 F12 控制台复制 console 输出即可.

**Q: 99脚本提示无法连接?**
A: 先在终端启动: `python e:\道\道生一\一生二\PCB设计\lceda_bridge\lceda_bridge_server.py`
