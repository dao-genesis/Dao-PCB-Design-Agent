# 仓库车间物流车控制系统设计 电气原理图设计说明

> 项目代号: `warehouse_logistics_vehicle`  ·  版本: v1.0  ·  创建: 2024-06-13  ·  更新: 2026-04-27

---

## 1. 项目概述

本系统为面向仓库车间物流场景的小型轮式物流车控制板. 以 STM32G030K8T6 为主控, 通过 L298N 驱动两个直流电机实现差速转向; 前向 HC-SR04 超声波测距用于避障; 底部双路红外循迹模块沿地面预设白线巡线; 站点红外探头识别站台标记触发停靠; OLED 屏显示运行状态; 蓝牙模块接收上位机指令并回传运行参数; 板载4个按键支持现场操作 (启停/速度/模式/复位); 压力接口预留为载重感知扩展; 电池电压通过分压 ADC 实时监测以触发低电量保护.

### 1.1 设计规格

- **电源输入**: 12V DC (锂电池供电)
- **主控**: STM32G030K8T6 (LQFP-32, ARM Cortex-M0+ @ 64MHz)
- **电机驱动**: L298N(XBLW) 双H桥, 2A/通道
- **无线通信**: 蓝牙 UART (HC-05/HC-04)
- **显示**: OLED 0.96" I2C (SSD1306)
- **传感器**: HC-SR04 超声波 + 红外循迹×2 + 站点红外 + 压力
- **电池监测**: ADC 分压采样 (R28 100K / R29 2K)
- **调试**: SWD (PA13/PA14) + 5V 程序下载
- **外形**: PCB 双面板 100×80mm (建议)

### 1.2 系统结构

系统由 **14 个功能模块** 组成, 共计 **50 个元器件**, **40 条电气网络**, **182 个引脚连接**.

## 2. 功能模块详细设计

### 2.1 12V转5V转3.3V稳压电路 (`power_chain`)

LR78M05D + AMS1117-3.3 双级线性稳压, 输入 12V 锂电池, 输出 5V 与 3.3V 双轨

**元件清单**:

- **U11** `LR78M05D` (TO-252-2) — 12V → 5V 一级稳压
- **U12** `AMS1117-3.3` (SOT-223) — 5V → 3.3V 二级稳压
- **C20** `100uF/25V` (C_Elec_5x5) — 12V 输入大电容滤波
- **C21** `10uF` (C0805) — 5V 输入端去耦
- **C22** `100nF` (C0603) — 5V 高频去耦
- **C23** `0.1uF` (C0603) — AMS1117 输入去耦
- **C24** `10uF` (C0805) — 3.3V 输出大电容稳压
- **D1** `LED-Green` (LED0603) — 状态指示
- **R21** `10K` (R0603) — D1 电源指示限流

**主要网络**: `12V`, `VCC_5V`, `VCC_3V3`, `GND`

**功能描述**:

- U11 LR78M05D: 12V → 5V/0.5A (TO-252)
- U12 AMS1117-3.3: 5V → 3.3V/1A (SOT-223)
- 去耦: C20=100uF, C21/C24=10uF, C22/C23=0.1uF
- D1 绿色LED + R21=10K — 3.3V 电源指示

### 2.2 电池电压ADC采集 (`battery_adc`)

R28(100K) / R29(2K) 高阻分压, 12V → 0.235V 落入 STM32 ADC 量程

**元件清单**:

- **R28** `100K` (R0603) — 电池电压上分压
- **R29** `2K` (R0603) — 电池电压下分压

**主要网络**: `VBAT`, `VBAT_S`, `GND`

**功能描述**:

- R28=100K  R29=2K
- 比例 1/51, 12V→235mV
- ADC 通道: PA0
- 建议加 100nF 滤波

### 2.3 超声波传感器接口 (`ultrasonic`)

HC-SR04 前向避障测距, Trig 触发 10us 脉冲, Echo 回波时长测距

**元件清单**:

- **U1** `HC-SR04` (PinHeader_1x4_P2.54mm) — 前向避障距离测量

**主要网络**: `VCC_5V`, `GND`, `US_TRIG`, `US_ECHO`

**功能描述**:

- U1 HC-SR04 (5V 供电)
- Trig: PB0 (10us 脉冲)
- Echo: PA4 (高电平时长 → 距离)
- 测距 2cm~400cm, 精度 ±3mm

### 2.4 OLED显示屏接口 (`oled`)

SSD1306 0.96" I2C OLED, 128×64 像素显示运行状态/电池/距离

**元件清单**:

- **OLED1** `SSD1306-128x64` (OLED_0.96"_4P) — 显示运行状态/电池电压/距离
- **R25** `4.7K` (R0603) — I2C SCL 上拉
- **R26** `4.7K` (R0603) — I2C SDA 上拉

**主要网络**: `VCC_3V3`, `GND`, `I2C_SCL`, `I2C_SDA`

**功能描述**:

- OLED1 SSD1306 128×64
- I2C: SCL=PB7, SDA=PB8
- 上拉 R25/R26 = 4.7K
- 刷新 ≤30Hz, 显示状态/电池/距离

### 2.5 复位单片机复位电路 (`reset`)

1NRST 复位按键 + R5 上拉 + C6 RC 滤波, 提供手动复位

**元件清单**:

- **RESET1** `TS-KG09S` (SW_TS-KG09S_6x6) — MCU 手动复位
- **R5** `10K` (R0603) — NRST 上拉
- **C5** `100nF` (C0603) — MCU VDD 高频去耦
- **C6** `1uF` (C0603) — NRST 滤波 + VDDA 去耦

**主要网络**: `NRST`, `VCC_3V3`, `GND`

**功能描述**:

- 1NRST 按键 (TS-KG09S)
- R5=10K 上拉 NRST
- C5=100nF MCU 去耦
- C6=1uF NRST 滤波

### 2.6 单片机引脚分配电路 (`mcu_core`)

STM32G030K8T6 LQFP-32 主控, 32 引脚分配如下表

**元件清单**:

- **U2** `STM32G030K8T6` (LQFP-32_7x7mm_P0.8mm) — 系统主控 — 传感器采样/通信/电机控制/显示

**主要网络**: `VCC_3V3`, `GND`, `NRST`, `SWCLK`, `SWDIO`

**功能描述**:

- U2 STM32G030K8T6 (LQFP-32, M0+ @ 64MHz)
- 电源: VDD/VDDA = 3.3V, 引脚 4/5
- SWD: PA13(SWDIO) / PA14(SWCLK/BOOT0)
- ADC: PA0(VBAT), PA2(压力)
- PWM: PB1(ENA), PB4(ENB) — TIM3
- 电机方向: PB2/3/5/6 → IN1/2/3/4
- I2C: PB7(SCL), PB8(SDA) — OLED
- USART: PA9(TX), PA10(RX) — 蓝牙
- 超声波: PB0(Trig), PA4(Echo)
- 循迹: PA6/PA7, 站点: PA1, LED: PA3

### 2.7 L298N电机驱动电路 (`motor_l298n`)

双 H 桥驱动两路直流电机, 含 8 个续流二极管 D2~D9 保护

**元件清单**:

- **U14** `L298N(XBLW)` (HSOP-20_L298N) — 左右电机正反转 + PWM 调速
- **D2** `1N5819` (SOD-123) — 电机续流保护
- **D3** `1N5819` (SOD-123) — 电机续流保护
- **D4** `1N5819` (SOD-123) — 电机续流保护
- **D5** `1N5819` (SOD-123) — 电机续流保护
- **D6** `1N5819` (SOD-123) — 电机续流保护
- **D7** `1N5819` (SOD-123) — 电机续流保护
- **D8** `1N5819` (SOD-123) — 电机续流保护
- **D9** `1N5819` (SOD-123) — 电机续流保护

**主要网络**: `12V`, `VCC_5V`, `GND`, `MOTOR_ENA`, `MOTOR_IN1`, `MOTOR_IN2`, `MOTOR_ENB`, `MOTOR_IN3`, `MOTOR_IN4`, `MOTOR_OUT1`, `MOTOR_OUT2`, `MOTOR_OUT3`, `MOTOR_OUT4`

**功能描述**:

- U14 L298N(XBLW) HSOP-20
- VS=12V (主电源), VSS=5V (逻辑)
- ENA/ENB: PWM 调速 (10kHz)
- IN1/2 — 左电机方向, IN3/4 — 右电机方向
- OUT1/2 — 左电机, OUT3/4 — 右电机
- D2~D9: 1N5819 续流二极管 (必装)
- 需散热片, 长时间满载加风扇

### 2.8 程序下载电路 (`swd`)

SWD 4 针接口: VCC_3V3 / SWCLK / SWDIO / GND, 双 100K 下拉防误进 ISP

**元件清单**:

- **PH-00015_SWD** `2.54mm-4P` (PinHeader_1x4_P2.54mm) — STM32 SWD 烧录与调试接口
- **R3** `100K` (R0603) — SWCLK 下拉
- **R4** `100K` (R0603) — SWDIO 下拉

**主要网络**: `VCC_3V3`, `GND`, `SWCLK`, `SWDIO`

**功能描述**:

- PH-00015 4P 排针
- R3/R4=100K 下拉
- 走线 < 100mm
- 兼容 ST-Link / DAP-Link

### 2.9 系统指示灯电路 (`sys_led`)

PA3 控制 LED6 (蓝色) 心跳指示, 1Hz 闪烁表示主循环运行

**元件清单**:

- **LED6** `LED-Blue` (LED0603) — 状态指示
- **R22** `10K` (R0603) — LED6 限流

**主要网络**: `VCC_3V3`, `GND`, `SYS_LED`

**功能描述**:

- LED6 蓝色, R22=10K 限流
- PA3 → LED6 → GND
- 1Hz 心跳, 故障时 5Hz

### 2.10 蓝牙模块驱动电路 (`bluetooth`)

HC-05 蓝牙串口模块, USART1 通信 (默认 9600bps)

**元件清单**:

- **BT1** `HC-05` (HC-05_Module) — 无线遥控 + 数据回传
- **LED5** `LED-Yellow` (LED0603) — 状态指示
- **R20** `470` (R0603) — LED5 蓝牙状态限流

**主要网络**: `VCC_3V3`, `GND`, `BT_UART_TX`, `BT_UART_RX`, `BT_LED_STATUS`

**功能描述**:

- BT1 HC-05 蓝牙2.0+EDR
- TX → PA10, RX ← PA9 (USART1)
- 默认 9600bps, AT 命令配置
- LED5 蓝牙状态指示, R20=470Ω
- 天线区净空 ≥10mm

### 2.11 按键控制电路 (`keys`)

4 个 TS-KG09S 轻触按键, 各配 10K 下拉, 软件去抖 20ms

**元件清单**:

- **KEY1** `TS-KG09S` (SW_TS-KG09S_6x6) — 用户输入按键
- **KEY2** `TS-KG09S` (SW_TS-KG09S_6x6) — 用户输入按键
- **KEY3** `TS-KG09S` (SW_TS-KG09S_6x6) — 用户输入按键
- **KEY4** `TS-KG09S` (SW_TS-KG09S_6x6) — 用户输入按键
- **R23** `10K` (R0603) — 按键1 下拉
- **R23B** `10K` (R0603) — 按键2 下拉
- **R23C** `10K` (R0603) — 按键3 下拉
- **R23D** `10K` (R0603) — 按键4 下拉

**主要网络**: `VCC_3V3`, `GND`, `KEY_1`, `KEY_2`, `KEY_3`, `KEY_4`

**功能描述**:

- KEY1: PA15 (启停)
- KEY2: PA9 (速度+)
- KEY3: PA10 (速度-)
- KEY4: PB9 (模式)
- R23×4 = 10K 下拉

### 2.12 压力采集接口 (`pressure`)

PH-00015 3P 接口, 对接 HX711 + 应变片或薄膜压力传感器

**元件清单**:

- **PH-00015_PRESS** `3P` (PinHeader_1x3_P2.54mm) — 载重感知扩展接口
- **R27** `10K` (R0603) — 压力 ADC 阻抗匹配 / 滤波

**主要网络**: `VCC_3V3`, `GND`, `PRESSURE_AOUT`

**功能描述**:

- 3P 排针: VCC_3V3/AOUT/GND
- AOUT → PA2 (ADC IN2)
- R27=10K 阻抗匹配
- 建议加 100nF RC 滤波

### 2.13 红外循迹模块 (`ir_trace`)

左右两路 TCRT5000 红外对管, 数字输出检测地面白线

**元件清单**:

- **IR_TRACE_L** `TCRT5000` (PinHeader_1x4_P2.54mm) — 左轮地面循迹
- **IR_TRACE_R** `TCRT5000` (PinHeader_1x4_P2.54mm) — 右轮地面循迹

**主要网络**: `VCC_3V3`, `GND`, `IR_LEFT_DO`, `IR_RIGHT_DO`

**功能描述**:

- IR_TRACE_L (左) → PA6
- IR_TRACE_R (右) → PA7
- TCRT5000 数字阈值可调
- 环境光强时建议加挡光罩

### 2.14 站点红外探头接口 (`ir_station`)

E18-D80NK 远距离红外, 检测站台反光标记触发停靠

**元件清单**:

- **IR_STATION** `E18-D80NK` (PinHeader_1x3_P2.54mm) — 识别站台反光标记

**主要网络**: `VCC_5V`, `GND`, `IR_STATION_DO`

**功能描述**:

- IR_STATION E18-D80NK
- 5V 供电, 检测距离 3~80cm
- DO → PA1, 高电平有效
- 软件计数 + 滤抖

## 5. 图纸设计说明

1. Q1/Q2 (实为 L298N 内部) 为 PWM 驱动桥臂, 频率建议 10~20kHz, 远离音频带
2. OLED 与 EEPROM 等 I2C 设备共用 PB6/PB7, 上拉电阻 4.7K 已配置
3. 电机驱动 12V 强电与 3.3V 控制信号必须分区铺地, 单点连 PGND ↔ SGND
4. 蓝牙天线区域板内净空 ≥10mm, 不可铺铜, 不可走信号线
5. SWD 接口须保留 PA13/PA14 + RESET + 3.3V + GND, 总长 < 100mm

## 6. 工程化注意事项

- ⚠️ 12V 进线必须加 SS54 反接保护二极管 + 自恢复保险丝 (建议 2A)
- ⚠️ L298N 散热片必须可靠贴装, 长时间满载需外加散热风扇
- ⚠️ 电池低于 9.0V 时蜂鸣报警并切断电机电源, 避免锂电池过放
- ⚠️ I2C 上拉与 OLED 共地必须可靠, 否则上电时序不稳定
- ⚠️ 红外循迹模块对环境光敏感, PCB 上可加挡光罩或采用调制红外方案
